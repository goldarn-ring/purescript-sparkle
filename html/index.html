FlareCheck

FlareCheck is a library which allows you to create interactive test suites in an easy way. The idea is to use the Flare library and the power of PureScripts type system to create interactive testing environments in an easy way.

This is in no way a replacement for properly written automated test suites but could be useful for certain scenarios (similar to manual tests in psci).

The library works in a similar way to QuickCheck, except that it creates reactive web interfaces to test the functions instead of using randomly generated data.

Examples

We want to create an interactive test suite for the following functions from the standard libraries:

30 charCodeAt :: Int -> String -> Maybe Int

Custom input fields

What if you want to test your own library with custom data types? Suppose we want to create a test for the following function from Data.String.Regex: 
20 match :: Regex -> String -> Maybe (Array (Maybe String))
FlareCheck provides the type class Flammable (analogous to QuickChecks Arbitrary): 
20 class Flammable a where
21   spark :: forall e. UI e a
where instances must provide a way to create a Flare UI which holds a value of type a.

To avoid orphan instances, we create a newtype wrapper and write a Flammable instance for Regex: 
13 newtype TRegex = TRegex Regex
14 
15 instance flammableTRegex :: Flammable TRegex where
16   spark = fieldset "Regex" $ TRegex <$>
17             (regex <$> string "Pattern" "fo+"
18                    <*> (parseFlags <$> string "Flags (g,i,m)" "g"))
Then, we can use flareCheck as before: 
30   flareCheck "match" $ \(TRegex regex) string -> match regex string
